What is Gridling?

Gridling is a lighweight CSS layout tool for building responsive websites fast.

Inspired by the Twitter Bootstrap framework, simple row and column classes are used to convert design layouts into a grid of halves, thirds, quarters and fifths. Gridling can be used on both small and larger projects, its system is adaptable and highly customisble for multiple scenarios.

See the demo's below for more about it's features.
